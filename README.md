# FingerPrint — 한글 지문자 실시간 인식 시스템

웹캠으로 한글 지문자(손으로 표현하는 자음·모음)를 실시간 인식하고, 인식된 자모를 완성형 한글 문장으로 조합하여 화면에 출력하는 프로그램입니다.

## 목차

1. 프로젝트 개요
2. 오픈소스 소개
3. 시스템 흐름
4. 깃허브 작업 과정
5. 실행 방법
6. 오픈소스 프로젝트로서의 장점

## 프로젝트 개요

청각·언어 장애인은 일상적인 의사소통에 수어를 사용하지만, 수어를 모르는 비장애인과의 소통에는 여전히 장벽이 있다. 본 프로젝트는 웹캠 한 대만으로 한글 지문자를 실시간 인식하고, 인식된 자모를 완성형 한글로 변환하여 1차적인 문자 소통을 돕는 것을 목표로 한다.

## 시스템 흐름

```
영상 입력 → 손 랜드마크 추출 → 특징 벡터 생성 → 자모 분류
        → 입력 안정화 → 한글 자모 조합 → 결과 출력
```

1. **영상 입력** — 웹캠 프레임을 받아 좌우 반전·색상 변환
2. **손 랜드마크 추출** — 손 관절 21개 좌표 검출
3. **특징 벡터 생성** — 관절 간 벡터·각도 계산, 10프레임 시퀀스 구성
4. **자모 분류** — 시퀀스를 모델에 입력해 31종 자모 중 하나로 예측
5. **입력 안정화** — 확률·연속 프레임 조건을 만족할 때만 자모 확정
6. **한글 자모 조합** — 확정 자모를 누적해 완성형 한글로 변환
7. **결과 출력** — 손 골격·인식 자모·완성 문장 표시, Backspace/ESC 제어

(상세 구조도는 발표 자료 참조)
